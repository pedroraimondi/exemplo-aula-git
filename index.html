Olá mundo novo

  1. Commit
    1. Check-point
    2. Save
    3. Guardar uma versão
    4. Passou por uma fase dificil e vai começar outra.
  2. Branch
    1. Cópia do seu código
    2. Cópia para fazer testes em cima
    3. Foto do quadro para poder testar um elemento novo
  3. Merge
    1. Juntar uma versão com a outra
    2. Mesclar
    3. Pegar o rascunho e passar a caneta
  4. Push
    1. Enviar o código para o repositório remoto
    2. Subir o código para um lugar onde todo mundo tem acesso
    3. Guardar no drive, oneDrive, etc...
    4. Deixar disponível na net...
  5. Pull
    1. Receber as atualizações remotas no local
    2. Pegar as coisas que estão na net e trazer para o meu computador
    3. Git fetch + Git merge
  6. Fetch
    1. Busca as atualizações remotas, mas não mergeia
    2. Atualiza mas não transfere as atualizações para sua branch local
    3. Não substitui o código
